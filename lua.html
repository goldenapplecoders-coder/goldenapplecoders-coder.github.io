Lua Programming • iAM Scholar

Lightweight scripting, game logic, and embedded programming with Lua

Course overview: Learn Lua from fundamentals to embedded scripting — covering syntax, tables, functions, metatables, and game engine integration. Ideal for game developers, embedded engineers, and automation designers.

Module 1 — Lua Basics

Setup, syntax, and writing your first Lua script.
Lesson 1 — Installing Lua and using the interpreter
Lesson 2 — Variables, types, and operators
Lesson 3 — Strings, numbers, and booleans
Lesson 4 — Basic input/output
Outcomes: Write and run basic Lua scripts with clean syntax.

Module 2 — Control Flow & Functions

Use conditionals, loops, and reusable functions.
Lesson 1 — if, elseif, else statements
Lesson 2 — while, repeat-until, and for loops
Lesson 3 — Defining and calling functions
Lesson 4 — Parameters and return values
Outcomes: Control logic and reuse code effectively with Lua functions.

Module 3 — Tables and Data Structures

Work with Lua’s flexible table system.
Lesson 1 — Creating and accessing tables
Lesson 2 — Iterating with pairs and ipairs
Lesson 3 — Nested tables and records
Lesson 4 — Table manipulation and sorting
Outcomes: Build dynamic data structures using Lua tables.

Module 4 — Metatables and OOP

Simulate object-oriented behavior using metatables.
Lesson 1 — Metatables and metamethods
Lesson 2 — index and inheritance
Lesson 3 — Encapsulation and constructors
Lesson 4 — Building class-like structures
Outcomes: Create reusable and modular Lua components.

Module 5 — Game Scripting and Capstone

Integrate Lua with game engines and build a final project.
Lesson 1 — Lua in Love2D and Roblox
Lesson 2 — Event-driven scripting
Lesson 3 — Debugging and performance tips
Lesson 4 — Capstone: design and present a Lua-powered game feature
Outcomes: Deliver interactive Lua scripts for games or embedded systems.
