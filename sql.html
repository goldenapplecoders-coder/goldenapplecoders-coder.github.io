SQL

SQL Programming • iAM Scholar

Relational databases, queries, joins, and data manipulation with SQL

Course overview: Learn SQL from fundamentals to advanced querying — covering schema design, joins, subqueries, indexing, and transactions. Ideal for data analysts, backend developers, and anyone working with relational databases.

Module 1 — SQL Basics

Introduction to databases and basic SQL syntax.
Lesson 1 — What is SQL and how databases work
Lesson 2 — SELECT, FROM, WHERE clauses
Lesson 3 — Filtering, sorting, and limiting results
Lesson 4 — SQL comments and formatting
Outcomes: Query and filter data from a single table.
SELECT name, age FROM students WHERE age > 18;

Module 2 — Data Manipulation

Insert, update, and delete records in a database.
Lesson 1 — INSERT INTO and VALUES
Lesson 2 — UPDATE and SET clauses
Lesson 3 — DELETE FROM and WHERE
Lesson 4 — Best practices for safe data changes
Outcomes: Modify data in relational tables securely.

Module 3 — Table Design and Relationships

Create tables and define relationships between them.
Lesson 1 — CREATE TABLE and column types
Lesson 2 — PRIMARY KEY and FOREIGN KEY
Lesson 3 — Normalization and schema design
Lesson 4 — ALTER TABLE and constraints
Outcomes: Design normalized schemas with relational integrity.

Module 4 — Joins and Subqueries

Combine data from multiple tables using joins and subqueries.
Lesson 1 — INNER JOIN, LEFT JOIN, RIGHT JOIN
Lesson 2 — CROSS JOIN and self joins
Lesson 3 — Subqueries in SELECT and WHERE
Lesson 4 — EXISTS and IN clauses
Outcomes: Write complex queries across multiple tables.

Module 5 — Aggregation and Grouping

Summarize and group data using aggregate functions.
Lesson 1 — COUNT, SUM, AVG, MIN, MAX
Lesson 2 — GROUP BY and HAVING clauses
Lesson 3 — Combining aggregates with joins
Lesson 4 — Window functions overview
Outcomes: Analyze and summarize data efficiently.

Module 6 — Transactions and Indexing

Control data integrity and optimize performance.
Lesson 1 — BEGIN, COMMIT, ROLLBACK
Lesson 2 — ACID properties and isolation levels
Lesson 3 — CREATE INDEX and performance tips
Lesson 4 — Capstone: design, query, and optimize
Outcomes: Build reliable and performant SQL applications.
BEGIN;
UPDATE accounts SET balance = balance - 100 WHERE id = 1;
UPDATE accounts SET balance = balance + 100 WHERE id = 2;
COMMIT;
