CSS • iAM Scholar

Practical CSS: layout systems, responsive design, components, performance, and design systems

Course overview

Summary: Focused CSS curriculum teaching selectors and specificity, modern layout with Flexbox and Grid, responsive patterns, component styling, CSS variables and theming, accessibility, performance, and building a small design system. Hands-on exercises, code examples, and a capstone.

Module 1 — CSS Fundamentals

Selectors, cascade, specificity, inheritance, and the box model.
Lesson 1 — Selectors & combinators, specificity table
Lesson 2 — Cascade, origin, importance, inheritance
Lesson 3 — Box model: content, padding, border, margin; box-sizing
Lesson 4 — Unit types: px, em, rem, %, vw/vh, relative sizing
Outcomes: Predict selector behavior, control layout primitives, size responsively.
/ specificity example /

header nav .link { color: red; } / high specificity /
.button { padding: 0.75rem 1rem; }

Module 2 — Visual Formatting & Typography

Control visual flow and create readable typography systems.
Lesson 1 — Display types, inline vs block, flow layout
Lesson 2 — Typography: font stacks, metrics, rhythm, vertical spacing
Lesson 3 — Color systems, contrast, and accessible palettes
Lesson 4 — Images, icons, SVG use and optimization
Outcomes: Build readable, accessible typographic systems and visual hierarchy.

Module 3 — Layout with Flexbox

Design one-dimensional layouts and responsive components with Flexbox.
Lesson 1 — Flex container properties: flex-direction, wrap, gap
Lesson 2 — Alignment: justify-content, align-items, align-self
Lesson 3 — Flex children: flex-grow, flex-shrink, flex-basis
Lesson 4 — Practical patterns: navs, toolbars, responsive stacks
Outcomes: Build resilient horizontal and vertical component layouts.
.row { display:flex; gap:1rem; align-items:center; }

Module 4 — Grid Layout

Two-dimensional layouts, grid template, areas, and responsive grids.
Lesson 1 — Grid fundamentals: tracks, gutters, explicit vs implicit
Lesson 2 — Template areas, repeat(), minmax(), auto-fit/auto-fill
Lesson 3 — Responsive grid patterns and masonry-like layouts
Lesson 4 — Combining Grid and Flexbox effectively
Outcomes: Solve complex layouts with Grid and fallback strategies.
.grid { display:grid; grid-template-columns: repeat(auto-fit, minmax(220px, 1fr)); gap:1rem; }

Module 5 — Responsive Design Patterns

Mobile-first, breakpoints, fluid design, responsive images and media queries.
Lesson 1 — Mobile-first approach and progressive enhancement
Lesson 2 — Responsive images: srcset, picture, sizes
Lesson 3 — Container queries overview and practical use
Lesson 4 — Designing for slow networks and low-spec devices
Outcomes: Build fluid interfaces that adapt gracefully to devices and network conditions.

Module 6 — CSS Variables, Theming & Design Tokens

Create reusable tokens, theme systems, and runtime theming with custom properties.
Lesson 1 — Declaring and scoping custom properties
Lesson 2 — Theming patterns and dark mode strategies
Lesson 3 — Design tokens and mapping to CSS variables
Lesson 4 — Runtime theming with JS and prefers-color-scheme
Outcomes: Build consistent, themeable UI systems with variables and tokens.
:root { --bg: #041224; --brand: #0ea5a4; }

Module 7 — Animations and Motion

Transitions, keyframe animations, performance-friendly motion, and reduced-motion support.
Lesson 1 — CSS transitions and transition timing functions
Lesson 2 — Keyframes and complex choreographies
Lesson 3 — Transform, opacity, and composite-only properties
Lesson 4 — prefers-reduced-motion and accessible motion design
Outcomes: Create performant animations that respect user preferences.
@media (prefers-reduced-motion: reduce) { .anim { animation: none !important; } }

Module 8 — Component Styling & Scoping

Patterns for reusable components: BEM, utility-first, CSS modules, shadow DOM basics.
Lesson 1 — Naming conventions: BEM and alternatives
Lesson 2 — Utility vs semantic CSS tradeoffs
Lesson 3 — Scoped styling: CSS modules and shadow DOM
Lesson 4 — Component accessibility patterns
Outcomes: Ship reusable, accessible components with predictable styling boundaries.

Module 9 — Performance, Critical CSS & Delivery

Optimize CSS for fast rendering and minimal blocking, critical CSS strategies.
Lesson 1 — Critical rendering path and minimizing render-blocking CSS
Lesson 2 — Splitting CSS, code-splitting, and async loading
Lesson 3 — Minification, purging unused CSS, and cache strategies
Lesson 4 — Using tools: Lighthouse, WebPageTest, CSS coverage
Outcomes: Improve initial paint and runtime performance by optimizing CSS delivery.

Module 10 — Design System Capstone

Build a mini design system: tokens, components, docs, and a pattern library.
Project brief — Produce a small design system with tokens, buttons, forms, and responsive layout components
Deliverables — Token JSON, CSS variable mapping, component examples, documentation site
Assessment — Consistency, accessibility, responsive behavior, and documentation quality
Outcomes: Create a reusable design system ready for integration into projects.

Deliverables — JSON token file, stylesheet, component demos, documentation site (static)
