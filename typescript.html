TypeScript • iAM Scholar

Typed JavaScript for safer, scalable applications — types, tooling, and modern patterns

Course overview

Summary: Practical TypeScript training covering types, compiler options, tooling, advanced type-level programming, frameworks, testing, and migration strategies from JavaScript. Hands-on labs, projects, and best practices for large codebases.

Module 1 — Getting Started with TypeScript

Install toolchain, configure tsconfig, and understand the TypeScript workflow.
Lesson 1 — Installing TypeScript, tsc, ts-node, and editor setup
Lesson 2 — tsconfig.json basics: target, module, strict, rootDir/outDir
Lesson 3 — TypeScript vs JavaScript: benefits and tradeoffs
Lesson 4 — Compiling, source maps, and debugging TS code
Outcomes: Compile TS projects, configure editor tooling, and run TS code locally.

Module 2 — Basic Types and Type Annotations

Primitive and composite types; annotate functions and variables.
Lesson 1 — Primitives: number, string, boolean, null, undefined
Lesson 2 — Arrays, tuples, enums
Lesson 3 — Any, unknown, never, void — when to use
Lesson 4 — Function types, optional and default params, rest parameters
Outcomes: Apply basic types and avoid unsafe any usage.
// example
function add(a: number, b: number): number {
  return a + b;
}

Module 3 — Advanced Types and Type Operators

Mapped types, conditional types, union/intersection, type inference techniques.
Lesson 1 — Union and intersection types; discriminated unions
Lesson 2 — Type guards, instanceof, in, custom guards
Lesson 3 — Mapped types and keyof, Record, Partial, Required
Lesson 4 — Conditional types and type inference patterns
Outcomes: Express complex type relationships and improve API ergonomics.

Module 4 — Generics and Reusable Types

Design generic utilities and reusable abstractions with type safety.
Lesson 1 — Function and class generics; generic constraints
Lesson 2 — Default type parameters and overload patterns
Lesson 3 — Utility types and composing generics (ReturnType, InstanceType)
Lesson 4 — Practical patterns: typed factories and builders
Outcomes: Build flexible, type-safe libraries and utilities.

Module 5 — Working with DOM, Browser APIs, and JSX

Type the DOM, events, and TypeScript in React and JSX contexts.
Lesson 1 — DOM typings: HTMLElement, Event types, querySelector generics
Lesson 2 — Typing fetch responses and async data
Lesson 3 — React + TypeScript: props, state, refs, and hooks typing
Lesson 4 — JSX intrinsic elements and custom element typings
Outcomes: Safely interact with the browser and type React components.

Module 6 — Tooling, Linters and Formatting

Integrate TypeScript with build tools, linters, and editors for a smooth developer experience.
Lesson 1 — ESLint + TypeScript integration and rule configuration
Lesson 2 — Prettier, format-on-save, and editor integrations
Lesson 3 — ts-node, ts-jest, and debugging TypeScript in Node and browsers
Lesson 4 — Path mapping, module resolution, and monorepo considerations
Outcomes: Maintain consistent style, reliable builds, and productive tooling.

Module 7 — Server-side TypeScript with Node

Type Node.js apps, databases, and API contracts using TypeScript.
Lesson 1 — tsconfig for Node, ESM vs CommonJS, runtime choices
Lesson 2 — Typing database models and ORMs (Prisma / TypeORM)
Lesson 3 — Typed API contracts with OpenAPI, zod, io-ts
Lesson 4 — Server testing, typed mocks, and fixtures
Outcomes: Deliver typed backend services with clear contracts and tests.

Module 8 — Advanced Type System Techniques

Type-level programming patterns used by libraries and frameworks.
Lesson 1 — Recursive and distributive conditional types
Lesson 2 — Template literal types and branded types
Lesson 3 — Declarative APIs: typed builders and DSLs
Lesson 4 — Authoring declaration files (.d.ts) and ambient modules
Outcomes: Read and author advanced type signatures and library types.

Module 9 — Migrating from JavaScript to TypeScript

Practical, low-risk strategies to adopt TypeScript incrementally.
Lesson 1 — Incremental adoption: allowJs, checkJs, and skipLibCheck
Lesson 2 — Typing third-party libraries and writing declaration files
Lesson 3 — Contract-first migration: interfaces, tests, and feature flags
Lesson 4 — Using codemods and automation to reduce manual work
Outcomes: Plan and execute a safe migration at any scale.

Module 10 — Testing, CI and Production Patterns

Testing strategies, runtime validation, and CI integration for TypeScript projects.
Lesson 1 — Unit and integration testing with Jest and ts-jest
Lesson 2 — Property-based testing and typed fixtures
Lesson 3 — Runtime validation with zod/io-ts and narrowing strategies
Lesson 4 — CI pipelines, type-check gates, and monitoring type drift
Outcomes: Ship TypeScript projects with strong test coverage and CI safeguards.

Capstone Project — Type-Safe Fullstack App

Design and ship a fullstack application with end-to-end types: API contracts, DB models, and frontend types.
Project brief — Build a small product (task manager, notes app) with typed API, typed DB layer, and React + TypeScript frontend
Deliverables — Repo, CI, tests, typed contracts, deployment link, demo video
Assessment — Type coverage, pragmatic API design, tests, documentation, and demo
Outcomes: Portfolio-ready, production-oriented TypeScript application.
