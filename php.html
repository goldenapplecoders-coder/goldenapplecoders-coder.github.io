PHP Programming • iAM Scholar

Dynamic web development, server-side scripting, and database integration with PHP

Course overview: Learn PHP from the ground up — covering syntax, forms, sessions, databases, and MVC architecture. Ideal for aspiring web developers, backend engineers, and CMS builders.

Module 1 — PHP Basics

Setup, syntax, and writing your first PHP script.
Lesson 1 — Installing PHP and setting up a local server
Lesson 2 — Variables, data types, and operators
Lesson 3 — Echo, print, and basic output
Lesson 4 — Comments and code structure
Outcomes: Run basic PHP scripts and understand server-side execution.

?>

Module 2 — Control Structures & Functions

Conditionals, loops, and reusable functions.
Lesson 1 — if, else, switch-case
Lesson 2 — for, while, and foreach loops
Lesson 3 — Declaring and calling functions
Lesson 4 — Parameters, return values, and scope
Outcomes: Control logic and reuse code effectively.

Module 3 — Forms and User Input

Handle form data and validate user input.
Lesson 1 — GET vs POST methods
Lesson 2 — $GET, $POST, and $_REQUEST
Lesson 3 — Input validation and sanitization
Lesson 4 — Error handling and feedback
Outcomes: Build interactive forms and process user data securely.

Module 4 — Sessions and Cookies

Manage user sessions and persistent data.
Lesson 1 — Starting and destroying sessions
Lesson 2 — $SESSION and $COOKIE usage
Lesson 3 — Authentication basics
Lesson 4 — Security tips for session management
Outcomes: Create login systems and manage user state.

Module 5 — PHP and MySQL

Connect PHP to databases and perform CRUD operations.
Lesson 1 — Connecting to MySQL with mysqli and PDO
Lesson 2 — SELECT, INSERT, UPDATE, DELETE queries
Lesson 3 — Prepared statements and SQL injection prevention
Lesson 4 — Displaying database results in HTML
Outcomes: Build dynamic web apps with persistent data.

Module 6 — MVC and Capstone

Organize code using MVC and complete a final project.
Lesson 1 — Introduction to MVC architecture
Lesson 2 — Building a simple MVC framework
Lesson 3 — Routing and controllers
Lesson 4 — Capstone: design, build, and present
Outcomes: Deliver a structured PHP web application and showcase your skills.
