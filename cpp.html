C++ Programming • iAM Scholar

Object-oriented programming, memory management, and modern C++ development

Course overview: Learn C++ from the ground up — covering syntax, classes, memory, templates, STL, file I/O, and modern C++ features. Ideal for aspiring software engineers, game developers, and systems programmers.

Module 1 — C++ Basics

Setup, syntax, and compiling your first program.
Lesson 1 — Installing a compiler and writing Hello World
Lesson 2 — Data types, variables, and operators
Lesson 3 — Input/output with cin/cout
Lesson 4 — Comments, formatting, and naming conventions
Outcomes: Compile and run basic C++ programs with standard I/O.
#include <iostream>

Module 2 — Control Flow & Functions

Conditionals, loops, and reusable functions.
Lesson 1 — if, else, switch-case
Lesson 2 — for, while, and do-while loops
Lesson 3 — Declaring and calling functions
Lesson 4 — Function overloading and default arguments
Outcomes: Control program logic and write reusable code blocks.

Module 3 — Object-Oriented Programming

Classes, objects, inheritance, and encapsulation.
Lesson 1 — Defining classes and creating objects
Lesson 2 — Constructors, destructors, and access modifiers
Lesson 3 — Inheritance and polymorphism
Lesson 4 — Virtual functions and abstract classes
Outcomes: Build modular, reusable, and extensible programs using OOP principles.

Module 4 — Memory Management

Pointers, dynamic memory, and smart pointers.
Lesson 1 — Pointer basics and memory addresses
Lesson 2 — Dynamic allocation with new/delete
Lesson 3 — Memory leaks and dangling pointers
Lesson 4 — Smart pointers: uniqueptr, sharedptr, weak_ptr
Outcomes: Manage memory safely and efficiently in C++ programs.

Module 5 — Templates and Generic Programming

Write reusable and type-safe code using templates.
Lesson 1 — Function templates and type deduction
Lesson 2 — Class templates and specialization
Lesson 3 — Variadic templates and parameter packs
Lesson 4 — Template metaprogramming basics
Outcomes: Build flexible and reusable components using C++ templates.

Module 6 — The Standard Template Library (STL)

Explore STL containers, algorithms, and iterators.
Lesson 1 — Vectors, lists, sets, maps, and stacks
Lesson 2 — Iterators and range-based loops
Lesson 3 — STL algorithms: sort, find, transform
Lesson 4 — Lambda expressions and functional patterns
Outcomes: Use STL to write efficient and expressive code.

Module 7 — File I/O and Streams

Read and write files using C++ streams.
Lesson 1 — Using fstream, ifstream, and ofstream
Lesson 2 — Reading and writing text and binary files
Lesson 3 — Stream formatting and manipulators
Lesson 4 — Error handling and file validation
Outcomes: Persist and retrieve data using file operations.

Module 8 — Modern C++ Features

Explore features from C++11 to C++20.
Lesson 1 — auto, nullptr, and range-based for loops
Lesson 2 — Move semantics and rvalue references
Lesson 3 — std::function, std::optional, and std::variant
Lesson 4 — Concepts and coroutines (C++20)
Outcomes: Write modern, expressive, and efficient C++ code.

Module 9 — Debugging, Testing & Tooling

Use tools to debug, test, and maintain
Lesson 1 — Debugging with gdb, Valgrind, and sanitizers
Lesson 2 — Writing unit tests with Google Test
Lesson 3 — Build systems: Makefiles, CMake, and Ninja
Lesson 4 — Static analysis, linters, and code formatters
Outcomes: Maintain high-quality C++ codebases with modern tooling and testing practices.

Module 10 — Capstone Project

Design and build a complete C++ application using all learned concepts.
Project brief — Build a console-based or GUI application (e.g., task manager, file parser, mini game engine)
Deliverables — Source code, documentation, test suite, and demo video
Assessment — Code structure, use of OOP/STL/templates, testing, performance, and presentation
Outcomes: Apply all concepts in a real-world C++ project and showcase your skills.

  Student s = {"Golden", 22};
  s.greet();

}
