HTML & CSS • iAM Scholar

Markup, semantics, responsive styling, layouts, accessibility, and modern CSS

Course overview: Practical HTML and CSS curriculum that teaches semantic markup, responsive layout techniques, accessibility, modern CSS (Flexbox, Grid), animations, and component styling. Each module contains lessons, code examples, small projects, and a capstone site build.

Module 1 — Foundations of HTML

Structure pages with semantic markup and understand the document tree.
Lesson 1 — HTML syntax, elements, attributes, and the DOM tree
Lesson 2 — Document structure: head, body, meta, title, charset
Lesson 3 — Semantic elements: header, nav, main, article, section, aside, footer
Lesson 4 — HTML forms: input types, labels, validation, accessibility considerations
Outcomes: Create accessible, semantic HTML pages and a basic form
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>Sample</title>
  </head>
  <body>
    <header><h1>Site Title</h1></header>
    <main><article><p>Content</p></article></main>
  </body>
</html>

Module 2 — Core CSS and Styling

Selectors, specificity, the cascade, colors, units, box model.
Lesson 1 — CSS syntax, selectors, combinators, specificity rules
Lesson 2 — Box model: margin, border, padding, content; box-sizing
Lesson 3 — Units, colors, typography and responsive font sizing
Lesson 4 — Developer tools, debugging layout and styles
Outcomes: Style pages with maintainable CSS and debug layout issues
/ example /
.container { max-width:1200px; margin:0 auto; padding:1rem; }
.card { background:#fff; border-radius:8px; box-shadow:0 6px 18px rgba(2,6,23,0.4); }

Module 3 — Layout: Flexbox and Grid

Responsive layouts using modern CSS layout systems.
Lesson 1 — Flexbox: axis concepts, alignment, wrapping
Lesson 2 — Grid: tracks, areas, template, auto-placement
Lesson 3 — Combining Grid and Flexbox for complex UIs
Lesson 4 — Responsive breakpoints, mobile-first patterns
Outcomes: Build responsive, accessible layouts without floats
.grid { display:grid; grid-template-columns: 1fr 320px; gap:1rem; }
@media (max-width:900px){ .grid{grid-template-columns:1fr;} }

Module 4 — Responsive Design & Mobile First

Designing UI that scales across devices and network conditions.
Lesson 1 — Viewport meta and mobile-first CSS
Lesson 2 — Fluid layouts, clamp(), min(), max()
Lesson 3 — Responsive images: srcset, picture, lazy loading
Lesson 4 — Progressive enhancement and graceful degradation
Outcomes: Build performant pages for mobile and desktop users
