Scala

Scala Programming • iAM Scholar

Functional programming, concurrency, and JVM integration with Scala

Course overview: Learn Scala from fundamentals to advanced functional programming — covering syntax, collections, OOP, concurrency, and Spark integration. Ideal for backend engineers, data engineers, and JVM developers.

Module 1 — Scala Basics

Setup, syntax, and writing your first Scala program.
Lesson 1 — Installing Scala and using REPL
Lesson 2 — Variables, types, and expressions
Lesson 3 — Functions and control flow
Lesson 4 — Basic I/O and formatting
Outcomes: Write and run basic Scala programs using idiomatic syntax.
object Hello extends App {
  println("Hello, iAM Scholar!")
}

Module 2 — Collections and Functional Programming

Work with lists, maps, and functional constructs.
Lesson 1 — Lists, sets, and maps
Lesson 2 — map, filter, reduce
Lesson 3 — Higher-order functions
Lesson 4 — Immutability and recursion
Outcomes: Build expressive and functional Scala code.

Module 3 — Object-Oriented Scala

Use classes, traits, and inheritance in Scala.
Lesson 1 — Classes and constructors
Lesson 2 — Traits and abstract members
Lesson 3 — Inheritance and polymorphism
Lesson 4 — Companion objects and case classes
Outcomes: Build modular and reusable Scala components.

Module 4 — Concurrency and Futures

Write concurrent and asynchronous Scala programs.
Lesson 1 — Threads and parallel collections
Lesson 2 — Futures and Promises
Lesson 3 — Error handling in async code
Lesson 4 — Akka basics and actors
Outcomes: Build scalable and concurrent systems with Scala.

Module 5 — Scala with Spark and Capstone

Use Scala for big data processing with Apache Spark.
Lesson 1 — Introduction to Spark and RDDs
Lesson 2 — DataFrames and transformations
Lesson 3 — Spark SQL and aggregations
Lesson 4 — Capstone: build and present a Scala + Spark project
Outcomes: Deliver production-ready data pipelines and showcase your skills.
