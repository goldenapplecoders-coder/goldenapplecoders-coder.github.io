Kotlin • iAM Scholar

Modern JVM and multiplatform development — concise, safe, and interoperable Kotlin.

Course overview

Summary: Practical Kotlin program covering language fundamentals, coroutines, JVM interop, Android basics, Kotlin Multiplatform, testing, and production patterns. Hands-on labs and a capstone project.

Module 1 — Kotlin Basics

Syntax, types, null-safety, and basic control flow.
Lesson 1 — Variables (val/var), basic types, type inference
Lesson 2 — Null safety: ?, !!, ?: and safe calls
Lesson 3 — Control flow: if, when, loops, expressions
Lesson 4 — Functions, top-level functions, default/ named params
Outcomes: Write idiomatic Kotlin with null-safety and concise syntax.

Module 2 — Object-Oriented & Functional Kotlin

Data classes, sealed classes, interfaces, and functional helpers.
Lesson 1 — Data classes, copy, equals, toString
Lesson 2 — Sealed classes and ADTs for safe modeling
Lesson 3 — Extension functions and properties
Lesson 4 — Higher-order functions and collection operations
Outcomes: Model domains clearly and use Kotlin's functional features.

Module 3 — Coroutines and Asynchrony

Structured concurrency with coroutines, flows, and common patterns.
Lesson 1 — Coroutine basics: launch, async, runBlocking
Lesson 2 — Dispatchers, coroutine scope, and cancellation
Lesson 3 — Flows: cold vs hot streams, operators
Lesson 4 — Error handling and testing coroutines
Outcomes: Implement efficient, cancellable async code using coroutines and flows.

Module 4 — JVM Interop & Tooling

Kotlin on the JVM, interop with Java, Gradle/Kotlin DSL and build tooling.
Lesson 1 — Java interop: @JvmStatic, @JvmOverloads, nullability mapping
Lesson 2 — Build with Gradle Kotlin DSL and project structure
Lesson 3 — Reflection, annotations, and kapt
Lesson 4 — Debugging, REPL and scripting with Kotlin
Outcomes: Integrate Kotlin into JVM ecosystems and build toolchains with Gradle.

Module 5 — Kotlin Multiplatform & Android Basics

Share code across platforms and get started with Android Kotlin development.
Lesson 1 — Kotlin Multiplatform concepts and project layout
Lesson 2 — Sharing business logic across JVM/JS/Native
Lesson 3 — Android basics: Activities, Jetpack Compose intro
Lesson 4 — Multiplatform testing and platform-specific bindings
Outcomes: Create reusable multiplatform modules and basic Android UIs with Compose.

Module 6 — Persistence & Networking

Database access, serialization, and HTTP clients for Kotlin apps.
Lesson 1 — Exposed/Room/SQLDelight basics for persistence
Lesson 2 — Serialization with kotlinx.serialization and Moshi/Gson
Lesson 3 — Ktor and Retrofit for HTTP clients and servers
Lesson 4 — Caching and offline patterns
Outcomes: Build resilient data layers and networked features.

Module 7 — Testing & Quality

Unit and integration testing, mocking, and CI for Kotlin projects.
Lesson 1 — JUnit 5 with Kotlin, assertions and fixtures
Lesson 2 — MockK for mocking and behavior verification
Lesson 3 — Instrumented Android tests and Robolectric
Lesson 4 — CI pipelines and quality gates for Kotlin projects
Outcomes: Ensure reliability with tests and continuous integration.

Module 8 — Advanced Patterns & Performance

Performance, memory, DSLs, and advanced language features.
Lesson 1 — Inline functions, reified generics, and performance tips
Lesson 2 — Writing Kotlin DSLs and type-safe builders
Lesson 3 — Memory profiling and allocation reduction
Lesson 4 — Interop patterns and migration strategies
Outcomes: Author high-performance Kotlin code and expressive DSLs.

Module 9 — Tooling, Libraries & Ecosystem

Explore popular Kotlin libraries, multiplatform libraries, and deployment considerations.
Lesson 1 — Ktor, kotlinx.coroutines, kotlinx.serialization
Lesson 2 — Jetpack Compose and Compose Multiplatform overview
Lesson 3 — Publishing artifacts and versioning
Lesson 4 — Community tools and learning resources
Outcomes: Navigate the Kotlin ecosystem and choose libraries wisely.

Module 10 — Capstone: Multiplatform App

Design and ship a small multiplatform application (shared logic + platform UI).
Project brief — Build a notes or task app with shared domain logic and platform UIs
Deliverables — Repo, tests, CI, platform builds (JVM/Android/JS), and README
Assessment — Code structure, tests, cross-platform reuse, and documentation
Outcomes: A portfolio-ready Kotlin multiplatform project.
