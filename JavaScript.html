JS

JavaScript • iAM Scholar

From fundamentals to modern frameworks, tooling, and production patterns — project-driven curriculum

Course overview: Learn JavaScript language fundamentals, modern ES features, DOM and browser APIs, Node.js backend, testing, build tooling, React basics, and deployable projects. Each module contains lessons, exercises, and a project.

Module 1 — JavaScript Fundamentals

Core language building blocks and runtime behavior
Lesson 1 — History and ecosystem; where JS runs
Lesson 2 — Syntax, variables (let/const/var), primitive types
Lesson 3 — Operators, expressions, template literals
Lesson 4 — Control flow: if/else, switch, loops
Outcomes: Write basic scripts, understand scoping and runtime quirks.

Module 2 — Functions, Scope, and Closures

Function patterns, lexical scope, and closures in practical use
Lesson 1 — Function declarations, expressions, arrow functions
Lesson 2 — Closures and private state
Lesson 3 — Hoisting and block scope
Lesson 4 — The arguments object and rest/spread
Outcomes: Build reusable functions and control scope safely.

Module 3 — Objects, Prototypes, and Classes

Object model, prototypes and modern class syntax
Lesson 1 — Object literals, property descriptors
Lesson 2 — Prototype chain and inheritance
Lesson 3 — ES6 classes, super, static methods
Lesson 4 — Object immutability and patterns
Outcomes: Model domain entities and inheritance patterns in JS.

Module 4 — Asynchronous JavaScript

Callbacks, promises, async/await and concurrency patterns
Lesson 1 — Event loop, microtasks vs macrotasks
Lesson 2 — Promises and chaining
Lesson 3 — async/await and error handling
Lesson 4 — Parallelism, throttling, debouncing
Outcomes: Write robust async code and reason about concurrency in the browser and Node.

Module 5 — DOM, Events and Browser APIs

Manipulate the DOM, handle events, and use browser-provided APIs
Lesson 1 — DOM traversal and manipulation
Lesson 2 — Event delegation, propagation, custom events
Lesson 3 — Web APIs: Fetch, Storage, History, Geolocation
Lesson 4 — Accessibility basics and progressive enhancement
Outcomes: Build interactive, accessible UIs with vanilla JS.

Module 6 — Modern ES and Tooling

Modern language features and developer tooling workflow
Lesson 1 — ES6+ features: destructuring, modules, optional chaining
Lesson 2 — Modules, bundlers (Rollup, Webpack, Vite) basics
Lesson 3 — Transpilation with Babel, polyfills, browser support
Lesson 4 — Linters, formatters, and editor integration (ESLint, Prettier)
Outcomes: Set up modern build pipelines and write portable ES code.

Module 7 — Node.js and Backend Fundamentals

Server-side JavaScript, HTTP, file I/O, and npm ecosystem
Lesson 1 — Node runtime, REPL, module resolution
Lesson 2 — Building servers with native http and Express
Lesson 3 — Working with databases (MongoDB / PostgreSQL) and ORMs
Lesson 4 — Authentication patterns, sessions and JWT
Outcomes: Build and deploy simple APIs and full-stack apps using Node.js.

Module 8 — Testing, Debugging and Quality

Write unit and integration tests, debug, and measure quality
Lesson 1 — Testing fundamentals with Jest or Mocha
Lesson 2 — DOM testing with Testing Library
Lesson 3 — Debugging in Chrome and VS Code; performance profiling
Lesson 4 — Code coverage, CI integration and healthy pipelines
Outcomes: Ship reliable code with tests and automated checks.

Module 9 — React Basics

Component-driven UI using React core patterns
Lesson 1 — React components, JSX, props and state
Lesson 2 — Hooks: useState, useEffect, custom hooks
Lesson 3 — Routing with React Router and code splitting
Lesson 4 — State management patterns (context, reducers)
Outcomes: Build SPA components and manage state predictably.

Module 10 — Advanced Frontend Patterns

Performance, accessibility, progressive web apps, and advanced UX patterns
Lesson 1 — Web performance optimization and critical rendering path
Lesson 2 — Service Workers and PWAs
Lesson 3 — Accessibility deep dive (ARIA, testing, semantics)
Lesson 4 — Advanced patterns: micro-frontends, module federation overview
Outcomes: Build high-performance, accessible, and resilient frontends.

Module 11 — Build and Deployment

Prepare apps for production and deploy to modern platforms
Lesson 1 — Bundling strategies, caching and long-term caching
Lesson 2 — Docker for frontend and backend apps
Lesson 3 — CI/CD pipelines (GitHub Actions), environment management
Lesson 4 — Serving strategies and CDNs
Outcomes: Deploy apps with repeatable pipelines and monitoring readiness.

Capstone Project

End-to-end project that ties together frontend, backend, testing, and deployment
Project ideas — Single page app with Node API and DB; Real-time chat app; Data dashboard with interactive charts
Deliverables — Source code, tests, CI config, Dockerfile, README, demo video
Assessment — Code quality, tests, UX, deployability and presentation
Outcomes: Portfolio-ready full-stack JavaScript application.
