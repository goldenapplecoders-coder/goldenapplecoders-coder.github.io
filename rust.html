Rust

Rust Programming • iAM Scholar

Safe systems programming, memory safety, and fearless concurrency with Rust

Course overview: Learn Rust from fundamentals to advanced systems programming — covering ownership, lifetimes, concurrency, traits, async, and deployment. Ideal for backend engineers, embedded developers, and performance-focused programmers.

Module 1 — Rust Basics

Setup, syntax, and writing your first Rust program.
Lesson 1 — Installing Rust and using Cargo
Lesson 2 — Variables, types, and mutability
Lesson 3 — Functions, expressions, and control flow
Lesson 4 — Modules and project structure
Outcomes: Build and run basic Rust programs using idiomatic syntax and Cargo.
fn main() {
  println!("Hello, iAM Scholar!");
}

Module 2 — Ownership and Borrowing

Understand Rust’s memory model and safety guarantees.
Lesson 1 — Ownership and move semantics
Lesson 2 — References and borrowing
Lesson 3 — Lifetimes and lifetime annotations
Lesson 4 — Copy types and smart pointers
Outcomes: Write memory-safe code without garbage collection.

Module 3 — Structs, Enums, and Pattern Matching

Model data and control flow with Rust’s type system.
Lesson 1 — Defining structs and methods
Lesson 2 — Enums and match expressions
Lesson 3 — Option, Result, and error handling
Lesson 4 — Destructuring and guards
Outcomes: Build expressive and safe data models with Rust.

Module 4 — Traits and Generics

Write reusable and abstract code with traits and generics.
Lesson 1 — Defining and implementing traits
Lesson 2 — Trait bounds and generic functions
Lesson 3 — Associated types and default implementations
Lesson 4 — Trait objects and dynamic dispatch
Outcomes: Build flexible and reusable abstractions in Rust.

Module 5 — Concurrency and Async

Write concurrent and asynchronous programs safely.
Lesson 1 — Threads and message passing
Lesson 2 — Mutexes, Arc, and shared state
Lesson 3 — async/await and futures
Lesson 4 — tokio and async runtimes
Outcomes: Build scalable and concurrent systems with Rust.

Module 6 — Testing and Tooling

Write tests and use Rust’s built-in tooling.
Lesson 1 — Unit tests and integration tests
Lesson 2 — cargo test, cargo bench, and coverage
Lesson 3 — Clippy, rustfmt, and linting
Lesson 4 — Debugging and profiling tools
Outcomes: Maintain high-quality Rust code with built-in tools.

Module 7 — Web APIs and Capstone

Build and deploy a Rust-based web service.
Lesson 1 — Building APIs with Actix Web or Axum
Lesson 2 — JSON, routing, and middleware
Lesson 3 — Dockerizing and deploying Rust apps
Lesson 4 — Capstone: design, build, and present
Outcomes: Deliver production-ready Rust services and showcase your skills.
use actix_web::{get, App, HttpServer, Responder};

[get("/")]
async fn hello() -> impl Responder {
  "Hello, iAM Scholar!"
}
