Go

Go Programming • iAM Scholar

Fast, concurrent, and scalable backend development with Go (Golang)

Course overview: Learn Go from fundamentals to production-grade services — covering syntax, concurrency, testing, web APIs, and deployment. Ideal for backend engineers, cloud developers, and systems programmers.

Module 1 — Go Basics

Setup, syntax, and writing your first Go program.
Lesson 1 — Installing Go and using go run/build
Lesson 2 — Variables, types, and functions
Lesson 3 — Packages, imports, and workspace layout
Lesson 4 — Basic I/O and fmt package
Outcomes: Build and run basic Go programs using idiomatic syntax.
package main

import "fmt"

func main() {
  fmt.Println("Hello, iAM Scholar!")
}

Module 2 — Control Flow & Functions

Conditionals, loops, and reusable functions.
Lesson 1 — if, else, switch
Lesson 2 — for loops and range
Lesson 3 — Function parameters and return values
Lesson 4 — Multiple return values and error handling
Outcomes: Write clean, reusable logic with Go functions and control structures.

Module 3 — Structs and Interfaces

Model data and behavior using Go’s type system.
Lesson 1 — Defining structs and methods
Lesson 2 — Interfaces and polymorphism
Lesson 3 — Composition and embedding
Lesson 4 — Custom types and type assertions
Outcomes: Build flexible and maintainable code using Go’s type system.

Module 4 — Concurrency with Goroutines

Write concurrent programs using goroutines and channels.
Lesson 1 — Goroutines and the go keyword
Lesson 2 — Channels and communication
Lesson 3 — Select statements and timeouts
Lesson 4 — Mutexes and sync primitives
Outcomes: Build safe and efficient concurrent programs in Go.

Module 5 — Testing and Tooling

Write tests and use Go’s built-in tooling.
Lesson 1 — Writing unit tests with the testing package
Lesson 2 — Table-driven tests and benchmarks
Lesson 3 — go fmt, go vet, and linting
Lesson 4 — Dependency management with go mod
Outcomes: Maintain high-quality Go code with built-in tools and tests.

Module 6 — Building Web APIs

Create RESTful services using Go’s net/http package.
Lesson 1 — HTTP handlers and routing
Lesson 2 — JSON encoding/decoding
Lesson 3 — Middleware and request context
Lesson 4 — Error handling and status codes
Outcomes: Build and expose APIs using Go’s standard libraries.

Module 7 — Deployment and Capstone

Ship your Go application and complete a final project.
Lesson 1 — Building binaries and cross-compilation
Lesson 2 — Dockerizing Go apps
Lesson 3 — Deploying to cloud platforms
Lesson 4 — Capstone: design, build, and present
Outcomes: Deliver production-ready Go services and showcase your skills.
// Sample handler
func hello(w http.ResponseWriter, r *http.Request) {
  fmt.Fprintln(w, "Hello, iAM Scholar!")
}
