Java • iAM Scholar

Object-oriented programming, JVM fundamentals, backend services, and enterprise patterns

Course overview

Summary: Complete Java curriculum covering core language, OOP, the JVM, concurrency, building RESTful services with Spring, testing, performance, and deployment. Project-driven with labs and a capstone.

Module 1 — Java Basics and Tooling

Install JDK, use javac/java, understand JRE vs JDK, set up IDE (IntelliJ/VS Code) and Maven/Gradle basics.
Lesson 1 — JDK setup, environment variables, SDKMAN
Lesson 2 — javac/jar/java commands and project layout
Lesson 3 — Build tools: Maven vs Gradle basics
Lesson 4 — Project structure and simple Hello World
Outcomes: Compile, run, and package Java apps; set up a project with Maven/Gradle.

Module 2 — Core Language and Types

Primitive and reference types, control flow, arrays, strings, and common APIs.
Lesson 1 — Primitives, boxing/unboxing, and wrapper classes
Lesson 2 — Strings, StringBuilder, formatting
Lesson 3 — Control flow, switch enhancements, and exceptions
Lesson 4 — Collections overview: List, Set, Map, Queue
Outcomes: Use Java core APIs and collections effectively.

Module 3 — Object-Oriented Design

Classes, interfaces, inheritance, composition, SOLID principles, and design patterns.
Lesson 1 — Classes, constructors, encapsulation, and immutability
Lesson 2 — Interfaces, abstract classes, default methods
Lesson 3 — SOLID principles and refactoring basics
Lesson 4 — Common patterns: Factory, Singleton, Strategy, Observer
Outcomes: Model domain problems with robust OOP and patterns.

Module 4 — Functional Java and Streams

Lambda expressions, functional interfaces, Stream API, and reactive introductions.
Lesson 1 — Lambdas and method references
Lesson 2 — Functional interfaces: Predicate, Function, Supplier, Consumer
Lesson 3 — Stream operations: map, filter, reduce, collectors
Lesson 4 — Intro to reactive streams and Project Reactor overview
Outcomes: Write expressive collection pipelines and understand reactive basics.
// example
List names = List.of("Ada", "Lin", "Sam");
List caps = names.stream().map(String::toUpperCase).collect(Collectors.toList());

Module 5 — Concurrency

Threads, executors, synchronization, concurrent collections, and common pitfalls.
Lesson 1 — Thread basics: Thread, Runnable, and lifecycle
Lesson 2 — ExecutorService, thread pools, and task submission
Lesson 3 — Synchronization: locks, volatile, atomic classes
Lesson 4 — Concurrency pitfalls: deadlocks, livelocks, and designing for safety
Outcomes: Build safe concurrent programs and choose appropriate concurrency primitives.

Module 6 — I/O, Serialization, and Persistence

File I/O, NIO, JSON/XML serialization, JDBC basics, and an intro to ORMs.
Lesson 1 — File I/O and NIO.2: Path, Files, and Streams
Lesson 2 — JSON with Jackson/Gson and XML processing
Lesson 3 — JDBC fundamentals and connection handling
Lesson 4 — Introduction to JPA/Hibernate and mapping patterns
Outcomes: Persist and serialize data reliably and safely.

Module 7 — Building RESTful Services with Spring Boot

Spring Boot fundamentals: project setup, controllers, DI, data access, and testing.
Lesson 1 — Spring Boot project setup, starters, and configuration
Lesson 2 — Controllers, request mapping, validation, and DTOs
Lesson 3 — Spring Data JPA, repository patterns, and transactions
Lesson 4 — Testing controllers and services with MockMvc and Testcontainers
Outcomes: Deliver maintainable REST APIs with Spring Boot and safety checks.

Module 8 — Testing, CI, and Quality

Unit tests, integration tests, mocking, and continuous integration pipelines.
Lesson 1 — JUnit 5 fundamentals and writing clear assertions
Lesson 2 — Mockito for mocking and behavior verification
Lesson 3 — Integration testing with Testcontainers and in-memory DBs
Lesson 4 — CI with GitHub Actions: build, test, and publish pipelines
Outcomes: Ensure code quality with automated tests and CI workflows.

Module 9 — Performance, Profiling & JVM Internals

Understand the JVM, profiling tools, GC behavior, and performance tuning techniques.
Lesson 1 — JVM architecture: class loading and memory layout
Lesson 2 — Garbage collectors and tuning basics
Lesson 3 — Profiling with VisualVM, async-profiler, and flame graphs
Lesson 4 — Optimizing hot paths and measuring improvements
Outcomes: Diagnose performance issues and apply safe JVM optimizations.

Module 10 — Distributed Systems & Cloud Deployments

Containerization, observability, microservice patterns, and deploying Java apps to cloud platforms.
Lesson 1 — Dockerizing Java apps and multi-stage builds
Lesson 2 — Service discovery, API gateways, and microservice patterns
Lesson 3 — Observability: metrics (Micrometer), logging, and tracing (OpenTelemetry)
Lesson 4 — Deploying to cloud platforms and autoscaling basics
Outcomes: Deploy and operate Java services with observability and resilience.

Capstone Project — Production-ready Java Service

Plan and build a full-featured backend service demonstrating API design, persistence, tests, CI, and deployment.
Project choices — Task manager API, inventory service, analytics ingestion pipeline
Deliverables — Source code, tests, Dockerfile, CI config, deployment link, architecture write-up
Assessment — Design clarity, test coverage, performance, documentation, and deployment
Outcomes: Portfolio-ready Java backend demonstrating end-to-end engineering skills.
