C# Programming • iAM Scholar

Modern C# development — object-oriented, asynchronous, and cross-platform with .NET

Course overview: Learn C# from fundamentals to advanced topics — covering syntax, OOP, LINQ, async programming, .NET libraries, and application development. Ideal for desktop, web, and cloud developers.

Module 1 — C# Basics

Setup, syntax, and writing your first C# program.
Lesson 1 — Installing .NET SDK and using Visual Studio Code
Lesson 2 — Basic syntax: variables, types, and operators
Lesson 3 — Console input/output and formatting
Lesson 4 — Comments, naming conventions, and code structure
Outcomes: Build and run basic C# programs using the .NET CLI.
