C

C Programming • iAM Scholar

Low-level programming, memory management, and systems development with C

Course overview: Learn C programming from scratch — covering syntax, memory, pointers, data structures, file I/O, and system-level programming. Ideal for aspiring systems engineers, embedded developers, and anyone who wants to master the fundamentals of computing.

Module 1 — Introduction to C

History, setup, compiling, and basic syntax.
Lesson 1 — Installing GCC/Clang and writing your first C program
Lesson 2 — Structure of a C program: headers, main(), return types
Lesson 3 — Compilation process: preprocessor, compiler, linker
Lesson 4 — Basic input/output with stdio.h
Outcomes: Compile and run basic C programs using terminal tools.
#include <stdio.h>

int main() {
  printf("Hello, iAM Scholar!\n");
  return 0;
}

Module 2 — Variables, Types & Operators

Primitive types, arithmetic, and type conversions.
Lesson 1 — int, float, char, double, and type modifiers
Lesson 2 — Operators: arithmetic, logical, bitwise
Lesson 3 — Type casting and implicit conversions
Lesson 4 — Constants and macros
Outcomes: Use variables and operators to perform calculations and logic.

Module 3 — Control Flow

Conditionals, loops, and switch statements.
Lesson 1 — if, else, and nested conditions
Lesson 2 — for, while, and do-while loops
Lesson 3 — break, continue, and goto
Lesson 4 — switch-case and enum usage
Outcomes: Control program logic with branching and iteration.

Module 4 — Functions and Scope

Function declarations, arguments, return types, and scope rules.
Lesson 1 — Declaring and calling functions
Lesson 2 — Scope: local, global, static
Lesson 3 — Recursion and stack behavior
Lesson 4 — Header files and modular programming
Outcomes: Write reusable functions and manage scope effectively.

Module 5 — Arrays and Strings

Working with collections of data and character sequences.
Lesson 1 — Declaring and accessing arrays
Lesson 2 — Multidimensional arrays and memory layout
Lesson 3 — String manipulation with char arrays
Lesson 4 — Common string functions: strlen, strcpy, strcat
Outcomes: Store and manipulate sequences of data and text.

Module 6 — Pointers and Memory

Mastering pointers, memory allocation, and referencing.
Lesson 1 — Pointer basics: declaration, dereferencing, and addresses
Lesson 2 — Pointer arithmetic and arrays
Lesson 3 — Dynamic memory: malloc, calloc, realloc, free
Lesson 4 — Common pitfalls: dangling pointers, memory leaks
Outcomes: Use pointers to manage memory and build dynamic structures.

Module 7 — Structures and Data Organization

Define and use custom data types with structs.
Lesson 1 — Declaring and accessing structs
Lesson 2 — Nested structs and arrays of structs
Lesson 3 — Passing structs to functions
Lesson 4 — Typedef and unions
Outcomes: Organize complex data using custom types.

Module 8 — File I/O and Streams

Read and write files using standard C libraries.
Lesson 1 — Opening, reading, writing, and closing files
Lesson 2 — Working with binary files
Lesson 3 — Error handling and file pointers
Lesson 4 — Formatted input/output with fprintf and fscanf
Outcomes: Persist and retrieve data using file operations.

Module 9 — Data Structures and Algorithms

Implement classic data structures and algorithms in C.
Lesson 1 — Linked lists: singly and doubly
Lesson 2 — Stacks and queues
Lesson 3 — Searching and sorting algorithms
Lesson 4 — Trees and recursion
Outcomes: Build efficient data structures and understand algorithmic logic.

Module 10 — System Programming and Capstone

Explore system-level programming and complete a final project.
Lesson 1 — Working with command-line arguments
Lesson 2 — Signals, processes, and system calls
Lesson 3 — Building a simple shell or text editor
Lesson 4 — Capstone project: design, build, and present
Outcomes: Apply all concepts in a real-world system-level project.
