Python • iAM Scholar

Beginner → Intermediate → Advanced • Data, Web, Automation • Project-driven curriculum

Course overview: A complete, hands-on Python program that starts with fundamentals and progresses to web development, data science, automation, and production-ready practices. Each module contains lessons, practical exercises, guided projects, and assessment rubrics.

Module 1 — Getting Started with Python

Set up your environment, understand Python basics, and write your first programs.
Lesson 1 — Installing Python and tools: CPython vs PyPy, pip, IDEs (VS Code), virtual environments (venv)
Lesson 2 — REPL, scripts, running programs, file structure
Lesson 3 — Basic syntax: variables, primitive types (int, float, bool, str), printing and formatting
Lesson 4 — Control flow: if, elif, else, truthy/falsy values
Outcomes: Install toolchain, run scripts, write simple programs; understand interpreter basics.

Module 2 — Data Structures and Core Language

Master Python built-in containers and core idioms used in real projects.
Lesson 1 — Lists and list methods, slicing, list comprehensions
Lesson 2 — Tuples and namedtuples
Lesson 3 — Dictionaries and dict methods, default dict, ordering
Lesson 4 — Sets and set operations
Lesson 5 — Strings, formatting methods, regex basics
Outcomes: Choose appropriate data structures, manipulate collections, use comprehensions and basic regex.

Module 3 — Functions, Modules, and Packages

Write modular, reusable code and learn packaging basics.
Lesson 1 — Defining functions, arguments, return values, docstrings
Lesson 2 — Scope, closures, lambdas, higher-order functions
Lesson 3 — Modules and packages, __name__ == "__main__", import patterns
Lesson 4 — Virtual environments, pip, requirements.txt, Poetry intro
Outcomes: Build modular code, publish/use packages locally, manage dependencies.

Module 4 — Error Handling, Testing, and Debugging

Make robust programs using exceptions, unit tests, and debugger tools.
Lesson 1 — Exceptions, try/except/else/finally, custom exceptions
Lesson 2 — Logging, assertions, defensive programming
Lesson 3 — Unit testing with unittest and pytest fundamentals
Lesson 4 — Debugging with VS Code and pdb; test-driven development (TDD) basics
Outcomes: Write tests, debug effectively, and handle runtime errors gracefully.

Module 5 — Object-Oriented Programming

Design classes and use OOP patterns for maintainable code.
Lesson 1 — Classes, instances, attributes, methods
Lesson 2 — Inheritance, composition, method overriding
Lesson 3 — Special methods (__init__, __repr__, __str__, __eq__)
Lesson 4 — Dataclasses and typing for classes
Outcomes: Model real-world problems using classes; use dataclasses for concise data models.

Module 6 — Functional Tools and Concurrency

Explore functional helpers and concurrency models for modern Python.
Lesson 1 — map, filter, reduce, itertools, functools
Lesson 2 — Generators, yield, generator expressions
Lesson 3 — Threading and multiprocessing basics
Lesson 4 — Asyncio: coroutines, event loop, async/await, practical examples
Outcomes: Use functional constructs and choose the right concurrency model for tasks.

Module 7 — File I/O, CSV, JSON, and Working with APIs

Read/write files and interact with remote services.
Lesson 1 — File operations, context managers, pathlib
Lesson 2 — CSV handling, JSON parsing and serialization
Lesson 3 — Requests, working with REST APIs, status codes and error handling
Lesson 4 — Consuming and paginating APIs, rate limiting strategies
Outcomes: Build ETL scripts and integrate with external APIs reliably.

Module 8 — Web Development with Flask (and intro to FastAPI)

Create web apps and APIs using Flask; overview of FastAPI for async APIs.
Lesson 1 — HTTP basics, routes, request/response lifecycle
Lesson 2 — Templates with Jinja2, static assets, form handling
Lesson 3 — REST API design, JSON responses, input validation
Lesson 4 — FastAPI intro: pydantic models, automatic docs, async endpoints
Outcomes: Deploy a Flask app and build production-ready APIs with FastAPI basics.

Module 9 — Data Analysis and Visualization

Use pandas, NumPy, and visualization tools to analyze data and tell stories.
Lesson 1 — NumPy arrays, vectorized operations
Lesson 2 — pandas DataFrame operations, grouping, joins
Lesson 3 — Data cleaning, missing data strategies
Lesson 4 — Visualization with Matplotlib and Seaborn; storytelling
Outcomes: Prepare and visualize real datasets; extract insights and summaries.

Module 10 — Machine Learning Fundamentals

Intro to supervised learning and practical ML workflows.
Lesson 1 — Scikit-learn basics: train/test split, pipelines
Lesson 2 — Linear models, decision trees, evaluation metrics
Lesson 3 — Feature engineering and model selection
Lesson 4 — Saving models, simple deployment patterns
Outcomes: Build and evaluate simple ML models and understand deployment considerations.

Module 11 — DevOps, Packaging, and Deployment

Prepare code for production and automate deployments.
Lesson 1 — Packaging with setuptools and Poetry
Lesson 2 — CI/CD basics with GitHub Actions (tests, lint, build)
Lesson 3 — Dockerizing Python apps; multi-stage builds
Lesson 4 — Monitoring, basic observability, and environment variables
Outcomes: Ship Python apps with reproducibility and CI/CD pipelines.

Module 12 — Security, Performance, and Best Practices

Harden applications and write efficient code with industry best practices.
Lesson 1 — Secure coding: input validation, avoiding injection, auth basics
Lesson 2 — Profiling and optimizing code; caching strategies
Lesson 3 — Type hints, static analysis (mypy, flake8), and linting
Lesson 4 — Licensing, open-source etiquette, code review practices
Outcomes: Produce secure, maintainable, and performant Python software.

Final Capstone Project

Design and ship a full-stack Python product combining modules learned.
Project choices — Data dashboard (Flask/FastAPI + pandas + charts); Web app with auth and DB; Automation pipeline that scrapes, transforms, and stores data.
Deliverables — Source code, README, tests, CI config, Dockerfile, deployment link, short demo video
Assessment — Code quality, tests, documentation, design decisions, and live demo
Outcomes: Real-world experience shipping a production-like Python service and portfolio-ready artifact.
